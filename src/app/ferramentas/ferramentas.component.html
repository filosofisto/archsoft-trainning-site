Ferramentas

Para este treinamento as ferramentas abaixo são necessárias.
* Requiridas

* Java 8 Update 261
* Git for Windows
* IntelliJ Community
* Player WRF
Apache Maven
  • Descompacte o Zip em uma pasta de sua preferência
  • Adicione a pasta <Maven Home>\bin no PATH do Sistema Operacional
  • Configure o IntelliJ para utilizar esta instalação de Maven
